❌ CAUSAS COMUNS DO ERRO

  1. API não responde: O arquivo api/certificados.php não está funcionando
  2. Banco de dados: Problemas de conexão com MySQL
  3. Erro de sintaxe: Problemas no código PHP
  4. Permissões: Arquivos sem permissão de execução
  5. Cache do navegador: Dados antigos em cache

🛠️ SOLUÇÕES ESPECÍFICAS

  1. Limpe o cache: Pressione Ctrl+F5 para forçar recarregamento
  2. Teste a API: Use o botão "Testar API de Certificados"
  3. Verifique o banco: Use o botão "Testar Conexão com Banco"
  4. Use dados simulados: Se o banco falhar, use dados de teste
  5. Reinicie o Apache: No XAMPP, pare e inicie o Apache novamente
