Instruções rápidas para acessar o sistema

🚀 SERVIDOR CONFIGURADO

Porta: 8080

URL Base: http://localhost:8080

✅ Passos para Acessar

  1. Inicie o XAMPP
    • Abra o XAMPP Control Panel
    • Clique "Start" no Apache
    • Clique "Start" no MySQL
  2. Teste a conexão
    • Abra: http://localhost:8080
    • Deve aparecer a página do XAMPP
  3. Acesse o sistema
    • Use os links abaixo

⚠️ Problemas Comuns

  • Erro de conexão: Verifique se o Apache está rodando no XAMPP
  • Porta ocupada: Feche outros programas que usem a porta 8080
  • Firewall: Permita o XAMPP no firewall do Windows
  • Permissões: Execute o XAMPP como administrador

🎯 Botões dos Relatórios

Os botões dos relatórios estão funcionais em:

  • Relatório de Cursos - Análise completa dos cursos
  • Relatório de Usuários - Estatísticas dos usuários
  • Relatório Financeiro - Análise financeira
  • Relatório de Agendamentos - Análise de aulas
