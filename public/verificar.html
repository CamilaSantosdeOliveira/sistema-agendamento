📋 Passos Detalhados:

  1. Localize o XAMPP Control Panel
    • Procure por "XAMPP" no menu iniciar
    • Ou vá para: C:\xampp\xampp-control.exe
  2. Inicie o Apache
    • Clique no botão "Start" ao lado de Apache
    • Aguarde até ficar verde
  3. Inicie o MySQL
    • Clique no botão "Start" ao lado de MySQL
    • Aguarde até ficar verde
  4. Teste a conexão
    • Abra: http://localhost:8080
    • Deve aparecer a página do XAMPP

🔍 Se ainda não funcionar:

  • Verifique se o XAMPP está instalado
  • Verifique se as portas 8080 e 3306 estão livres
  • Execute o XAMPP como administrador
  • Verifique o firewall do Windows
