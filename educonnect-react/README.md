# EduConnect React

Sistema de agendamento educacional desenvolvido com React, CSS próprio, PHP e MySQL.

## Tecnologias

- React
- JavaScript
- CSS próprio
- PHP
- MySQL
- Vite
- Git

## Como rodar

```bash
npm install
npm run dev
```

Acesse:

```text
http://localhost:3002
```

## Backend

A versão React foi criada para evoluir o projeto atual sem apagar os arquivos existentes. A integração com PHP/MySQL pode reaproveitar as APIs já presentes no projeto principal.
